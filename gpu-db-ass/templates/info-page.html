 GPU-

 Vram-

 VRAM (Video Random Access Memory) is specialized memory used by a graphics card to store image data and textures for fast access and rendering in visual applications.

 clockspeed in MHz-

 Clock speed in GPUs refers to the frequency at which the graphics processing unit operates, determining how quickly it can process instructions and perform calculations.
